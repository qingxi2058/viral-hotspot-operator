# 爆款选题操盘手

把一个热点或选题，快速变成可拍、可发、可改写的短视频内容包。

追了热点没播放，不追又怕错过。问题通常不是热点本身，而是切错了角度。

这个 Skill 不是单纯帮你“写标题”，也不是机械“追热点”。它会先确认事实边界，再判断这个热点值不值得跟，然后帮你选角度、做标题、定开头，最后收成一版可以直接拿去拍的视频内容方案。

## 这个 Skill 适合谁

- 想追热点但总担心追错的人
- 需要快速出视频选题的内容创作者
- 想把热点做成爆款切口的运营和操盘手
- 想要更快产出短视频内容包的个人博主和团队

## 它能做什么

- 先确认热点来源、发生时间和事实边界
- 从时效性、关联性、传播性、可视化、延展性 5 个维度快速评分
- 只给三种判断：值得立刻跟 / 值得跟但要换角度 / 不建议跟
- 给出 3 个可切角度，并推荐 1 个主打法
- 生成 8 到 12 个标题候选，精选 Top 3，再定主标题
- 生成 3 秒开头钩子
- 按平台输出 30 到 90 秒视频内容结构
- 给出封面文案、拍摄提纲、字幕提示和评论区引导
- 补充多平台改写建议

## 典型使用场景

- “这个热点能不能跟？”
- “这个选题能不能做？”
- “帮我把这条新闻做成短视频。”
- “给我一版更容易点开的标题。”
- “这个事件应该从哪个角度切？”
- “我想 10 分钟内产出一版可拍内容包。”

## 推荐输入格式

为了让输出更准，建议这样给信息：

```text
1. 热点是什么：
2. 链接/截图/来源：
3. 你的账号方向：
4. 想发平台：小红书 / 抖音 / 视频号
5. 目标：播放 / 涨粉 / 转化 / 建立人设
```

## 输出长什么样

默认输出：

1. 事实确认
2. 值不值得跟
3. 推荐主标题
4. 备选标题 Top 3
5. 封面文案
6. 3 秒开头钩子
7. 3 个可切角度和推荐主打方向
8. 30 到 90 秒视频内容结构
9. 拍摄提纲
10. 关键字幕/画面提示
11. 评论区引导句
12. 多平台改写建议

判断结论只给三种：

- 值得立刻跟
- 值得跟，但要换角度
- 不建议跟，给替代选题

## 能力边界

这个 Skill 不负责：

- 自动发布
- 长文写作
- 机械搬运热点
- 为了追热点而追热点
- 把未确认的信息写成事实

如果一个热点根本不值得做，它会直接告诉你别做，并给你更好的替代切法。如果一个热点已经过了正面追的窗口，但还能做复盘、案例或趋势判断，它会建议你换角度，而不是硬追。
