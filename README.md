# 爆款热点操盘手

把一个热点，快速变成可拍、可发、可改写的短视频内容包。

这个 Skill 不是单纯帮你“写标题”，也不是机械“追热点”。它会先判断这个热点值不值得跟，再帮你选角度、做标题、定开头，最后收成一版可以直接拿去拍的视频内容方案。

## 这个 Skill 适合谁

- 想追热点但总担心追错的人
- 需要快速出视频选题的内容创作者
- 想把热点做成爆款切口的运营和操盘手
- 想要更快产出短视频内容包的个人博主和团队

## 它能做什么

- 判断一个热点值不值得跟
- 给出 3 个可切的角度，并推荐 1 个主打法
- 生成更抓人的标题和开头钩子
- 输出 30 到 60 秒的短视频内容结构
- 给出封面文案、拍摄提纲、字幕提示和评论区引导
- 补充多平台改写建议

## 典型使用场景

- “这个热点能不能跟？”
- “帮我把这条新闻做成短视频。”
- “给我一版更容易点开的标题。”
- “这个事件应该从哪个角度切？”
- “我想 10 分钟内产出一版可拍内容包。”

## 输出长什么样

默认输出：

1. 推荐主标题
2. 备选标题 Top 3
3. 封面文案
4. 3 秒开头钩子
5. 30 到 60 秒内容结构
6. 拍摄提纲
7. 关键字幕/画面提示
8. 评论区引导句
9. 多平台改写建议

## 能力边界

这个 Skill 不负责：

- 自动发布
- 长文写作
- 机械搬运热点
- 为了追热点而追热点

如果一个热点根本不值得做，它会直接告诉你别做，并给你更好的替代切法。
